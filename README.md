# Social Media Addiction Analytics Dashboard

### Setup

First, clone the repository and navigate into the project directory:

``` bash
git clone git@github.com:UBC-MDS/DSCI-532_2026_30_social-media-addiction.git
cd DSCI-532_2026_30_social-media-addiction/
```

Then install the required R packages from within an R console:

``` r
install.packages(c("shiny", "bslib", "dplyr", "ggplot2", "plotly", "readr"))
```

To run the shiny app locally, run the following command from the root of the project:

``` bash
shiny::runApp('src')
```

You will be able to access the app at the link displayed in the terminal.
